# Cosign

Cosign is a full stack social media platform built with **Django (web and API)** and **React Native (mobile)**, designed to reimagine online interaction and entertainment in a way thatâ€™s **healthier, safer, and more community driven** over the long term.

Instead of infinite feeds and engagement maximizing algorithms, Cosign is inspired by apps like **BeReal** and earlier eras of **Instagram/Facebook**, emphasizing **authentic community, comradery, and intentional sharing** as opposed to addiction and rivalry.

---

## Project Goals

- Promote **healthier usage patterns** (less doomscrolling / infinite content)
- Prioritize **community and real connections**
- Build a platform designed around **safety and long term user well being**

---

## Features (high level)

### Core Social
- User accounts paired with authentication
- Profiles (bio, avatar, etc.)
- Create and share posts that have time sensitive public feed accessibility
- Social interactions (likes/comments/follows)

### Healthier Content Experience
- No infinite content loop by default
- Intentional sharing model inspired by BeReal / early social platforms
- Community first design principles

### Cross-Platform
- **Web:** Django-powered interface
- **Mobile:** React Native app for iOS/Android
- **Backend:** Django API powering both clients

---

## Tech Stack

- **Backend:** Django (Python), Django ORM, Django REST Framework (REST API)
- **Web:** Django templates
- **Mobile:** React Native
- **Database:** SQLite
